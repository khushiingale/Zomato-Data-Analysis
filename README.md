# 🍽️ Zomato Data Analysis

## 🚀 Overview
This project focuses on analyzing Zomato restaurant data to extract meaningful insights about customer preferences, restaurant trends, and ratings. It helps understand factors affecting restaurant success using data visualization and analysis.

---

## 🎯 Features
- 📊 Data cleaning and preprocessing  
- 📈 Exploratory Data Analysis (EDA)  
- ⭐ Rating and review analysis  
- 🌍 Location-based insights  
- 💰 Cost vs rating comparison  
- 📉 Visualizations for better understanding  

---

## 🛠️ Tech Stack
- Python 🐍  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## 📦 Installation & Setup

```bash
pip install pandas numpy matplotlib seaborn
```

## ⚙️ How It Works
1. Load Zomato dataset
